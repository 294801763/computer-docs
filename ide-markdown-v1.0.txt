# Markdown - CSS 设置
文件->设置->语言和框架->Markdown->自定义CSS->规则
<br/>
`--------------------`
```css
/*段落*/
p
{
	font-size: 14px;
	margin: 0px;
	padding:0px;
}

/*标题*/
h1 , h2 , h3 , h4 , h5 , h6
{
	font-size: 14px;
	margin: 0px;
	padding:0px;
}

/*代码段*/
pre {
	margin: 0px;
	padding:0px;
	font-size: 14px;
}

/*行内代码*/
code {
	font-family: 'Courier New', Courier, monospace;
	font-size: 14px;
	color: #000000; 
	margin: 0px;
	padding: 0px;
	border: 0px;
	border-radius: 3px;
}

/*分割线*/
hr {
	border: 1px dashed #000000;
	height: 1px;
	margin: 0px;          
	padding:0px;
	background: #333;
	background-image: linear-gradient(to right, #ccc, #333, #ccc);
	box-shadow: 0 2px 2px rgba(0, 0, 0, 0.5); /* 阴影效果 */
}
```

# 插入代码段
```c
#include <stdio.h>
#include <version.h>

int main(int argc,char **argv)
{
   // 输出当前程序的版本信息到控制台
   printf("Version: %s\n", VERSION);

  // 显式标记参数 argc 为未使用，避免编译器警告
  UNUSED(argc) ;

  // 显式标记参数 argv 为未使用，避免编译器警告
  UNUSED(argv);

  return 0;
}
```

  function toggleContent() {
    var content = document.getElementById('hidden-content');
