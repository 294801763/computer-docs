器件移动,旋转,镜像,对齐,等间距
---------------
Edit->Move                  移动 

options含义
-----
Ripup etch                 移动对象时去除所连走线,过孔
Slide etch                 移动对象时保留所连走线,过孔,走线随着对象平滑移动
Stretch etch               移动对象时保留所连走线,过孔,走线随着对象以任意角度移动
当以上三项都不勾选时,表示仅移动对象,不能影响其他走线,过孔 

Type                       以相对坐标或绝对坐标的方式旋转,一般选相对坐标
Angle                      旋转的角度设置,根据实际需要选择,一般90度
Point                      设置旋转时的基准点,有以下几种方式,根据实际情况进行选择
 Sym origin                以元器件封装原点为基准点,软件默认该选项
 Body center               以元器件place_bound几何中心为基准,常用于元器件原地旋转
 User pick                 以鼠标单击选择点为基准,常用多个元器件的整体旋转
 Sym pin                   以元器件引脚编号为基准点,常用于元器件结构定位
Dynamic Alignment          点击Preferences进入User Preferences进行设置勾选,可以在移动器件时与其它器件进行对齐操作

旋转
-----
Edit->Spin                 spin命令,当PCB右下角显示了spin时表示正在执行命令,此时同样可以在find面板和option面板中进行相关设置,参数设置与移动命令设置完全一样
rotate                     rotate命令,不能单独使用,需要先执行move命令,选中器件之后,点击鼠标右键选择rotate命令进行旋转,同样可以在find面板和option面板中进行相关设置参数设置与移动命令设置完全一样

镜像
-----
Edit->Mirror               在布局的时候,需要将顶层器件布局至底层时就需要用到mirror命令,当pcb界面右下角显示mirror时表示正在执行命令,同时在find面板上勾选Symbol或者Groups,然后单击一下器件就完成操作,也可以在移动器件命令菜单下,点击鼠标右键选择镜像命令进行操作

Capture与PCB Editor同步网表与错误解析
---------------
设置设计单位,Step->Design Parameters...->Design->Command parameters->Size->User units(设计单位):Mils

step.1 创建网表
-----
1.1 选择<name>.dsn,Tools->Create Netlist->PCB Footprint:PCB Footprint,Create PCB Editor Netlist->Options->Netlist Files:<填写路径>\allegro->OK/确定
网表的常见错误:1.管脚不匹配<多适用于第一方网表(Cadence)>,2.找不到Device文件<多适用于第三方网表>,3.没有Flash文件

step.2 网表的使用
-----
2.1 指定封装库,Setup->User Preferences...->Paths->Library->padpath:<.pad文件路径>,psmpath:<.psm文件路径>
2.2 创建PCB工程,File->New...->Drawing Name:<name>,Drawing Type:Board->OK/确定 
2.3 导入网表,File->Import->Logic/Netlist...->Cadence->Import logic type->Design Entry CIS(Capture)                    
                                           ->Place changed component->Always
									       ->Import directory:<选择上面生成的网表路径即可>

导入网表之后,PCB编辑区域并没有任何内容.
Display->Status...->Symbols and nets[重点关注<Unplaced symbols,Unrouted nets,Unrouted connections>]

2.4 放置元器件,Place->Quickplace...->Place by property/value
                                   ->Placement position->By user pick  
								   ->Board layer:Top 
								   ->Place(放置)

2.5 显示/关闭飞线 
Display->Show Rats->All(打开飞线)
Display->Blank Rats->All(关闭飞线)

Display->Status...->Symbols and nets[重点关注<Unplaced symbols,Unrouted nets,Unrouted connections>]

2.6 放置单个器件
Logic->Part Logic...->Part Selection Area-><器件列表>-><选择一个并更改其位号>->Part Modification Area->Refdes:<填一个唯一的名称即可>->Add 
Place->Quickplace...->Place by property/value
                                   ->Placement position->By user pick  
								   ->Board layer:Top 
								   ->Place(放置)

2.7 分配网络
Logic->Net Logic
Options->Net:<选择网络名称>
接下来选择器件的Pin(焊盘)即可

2.8 器件的删除 
Find->All Off
    ->勾选:Symbols

PCB Editor软件网络飞线常规处理 
---------------	
1.1 关闭飞线,Display->Blank Rats->All(关闭飞线)
1.2 Find->All Off
    ->勾选:Nets
1.3 显示网络飞线,Display->Show Rats->Net,然后选择焊盘上的网络名即可
1.4 关闭飞线,,Display->Blank Rats->All(关闭飞线)
1.5 显示网络飞线,Display->Show Rats->Net,然后选择焊盘上的网络名即可
1.6 关闭飞线,,Display->Blank Rats->All(关闭飞线)

注意:周而复始即可

1.7 Find->All Off
    ->勾选:Nets
1.8 编辑GND的属性,隐藏GND飞线,Edit->Properties然后选择焊盘上的NET(GND)在打开的对话框中选择"Voltage","Value":0,应用(Apply)之后再打开飞线就可以看到GND的飞线被隐藏了
1.9 编辑VCC 3_3V的属性,隐藏VCC 3_3V飞线,Edit->Properties然后选择焊盘上的NET(VCC 3_3V)在打开的对话框中选择"Voltage","Value":3.3,应用(Apply)之后再打开飞线就可以看到VCC 3_3V的飞线被隐藏了
1.10 编辑GND的属性,显示GND飞线,Edit->Properties然后选择焊盘上的NET(GND)在打开的对话框中选择"Voltage",勾选Delete->Voltage,应用(Apply)之后再打开飞线就可以看到GND的飞线被隐藏了
1.11 编辑VCC 3_3V的属性,显示VCC 3_3V飞线,Edit->Properties然后选择焊盘上的NET(VCC 3_3V)在打开的对话框中选择"Voltage",勾选Delete->Voltage,应用(Apply)之后再打开飞线就可以看到VCC 3_3V的飞线被隐藏了
