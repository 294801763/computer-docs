使用 ImageSearch 识别按钮并点击
-------------------
如果按钮是一个图像,可以通过 ImageSearch 命令找到按钮的位置,然后使用 MouseClick 或 Click 命令进行点击.
CoordMode, Pixel, Screen  ; 设置坐标模式为屏幕坐标
CoordMode, Mouse, Screen  ; 设置鼠标移动的坐标模式为屏幕坐标

Loop {
    MouseGetPos, OriginalX, OriginalY  ; 获取当前鼠标位置
    if ImageSearch(FoundX, FoundY, 0, 0, A_ScreenWidth, A_ScreenHeight, "C:\path\to\button.png") {
        MouseMove, FoundX, FoundY  ; 将鼠标移动到按钮位置
        MouseClick, Left  ; 点击按钮
        MouseMove, OriginalX, OriginalY  ; 恢复鼠标位置
    }
    Sleep, 500  ; 等待一段时间后再次检查
}

说明:
ImageSearch   用于在屏幕上搜索指定的图像(按钮)
MouseClick    或 Click 用于模拟鼠标点击
CoordMode     设置坐标模式为屏幕坐标,确保搜索和点击操作基于整个屏幕

使用 ControlClick 点击窗口中的按钮
-------------------
如果按钮是一个窗口控件,可以通过 ControlClick 命令直接点击按钮,无需知道按钮的具体位置.

; 点击标题为“窗口标题”的窗口中的“按钮文本”按钮
ControlClick, 按钮文本, 窗口标题

说明:
ControlClick   可以通过控件的文本或类名直接点击按钮。
如果按钮是标准的 Windows 控件，这种方法非常高效。

使用 Acc 库点击网页或其他复杂界面中的按钮
-------------------
对于网页或其他复杂界面中的按钮,可以使用 Acc 库(Accessibility API)来识别和操作按钮.

; 假设按钮的 ID 已知
ID_按钮 := "4.1.1.4.1.3.5"
WinGet, hWnd, ID, A  ; 获取当前窗口的句柄
oAcc := Acc_Get("Object", ID_按钮, 0, "ahk_id " hWnd)
if oAcc {
    oAcc.accDoDefaultAction(0)  ; 执行按钮的默认动作（点击）
}

说明:
Acc_Get              用于获取指定 ID 的控件对象
accDoDefaultAction   用于执行控件的默认动作(如点击按钮)
这种方法适用于复杂的界面,如网页或自定义控件.

使用 PixelGetColor 识别颜色并点击
-------------------
如果按钮的颜色是唯一的,可以通过 PixelGetColor 检测按钮位置并点击.

CoordMode, Pixel, Screen  ; 设置像素坐标模式为屏幕坐标

Loop {
    MouseGetPos, OriginalX, OriginalY  ; 获取当前鼠标位置
    if PixelGetColor, color, 100, 100, RGB  ; 获取按钮位置的颜色
    {
        if color = 0xRRGGBB  ; 比较颜色值（按钮的颜色）
        {
            MouseMove, 100, 100  ; 移动鼠标到按钮位置
            MouseClick, Left  ; 点击按钮
            MouseMove, OriginalX, OriginalY  ; 恢复鼠标位置
        }
    }
    Sleep, 500  ; 等待一段时间后再次检查
}

说明：
PixelGetColor   用于获取指定位置的颜色.
通过比较颜色值来判断按钮是否存在.

自动登录桌面应用程序
-------------------
如果需要自动登录桌面应用程序(如:QQ,微信等),可以通过模拟键盘输入和鼠标点击来实现.

; 启动 QQ
Run, "C:\Program Files (x86)\Tencent\QQ\Bin\QQ.exe"  ; 修改为你的 QQ 安装路径
WinWait, QQ  ; 等待 QQ 窗口出现
WinActivate, QQ  ; 激活 QQ 窗口

; 输入账号和密码
Sleep, 2000  ; 等待窗口加载完成
Send, your_qq_account  ; 输入账号
Send, {Tab}  ; 切换到密码框
Send, your_qq_password  ; 输入密码
Send, {Enter}  ; 按回车键登录

; 启动微信
Run, "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"  ; 修改为你的微信安装路径
WinWait, 微信  ; 等待微信窗口出现
WinActivate, 微信  ; 激活微信窗口

; 模拟点击登录按钮（需要根据实际位置调整坐标）
Sleep, 2000
MouseClick, Left, 500, 300  ; 点击登录按钮

自动登录网页
-------------------
自动登录网页时,可以通过 ComObjCreate 创建IE浏览器对象,并通过DOM操作输入账号和密码.

; 打开网页
oIE := ComObjCreate("InternetExplorer.Application")
oIE.Visible := True
oIE.Navigate("https://example.com/login")  ; 替换为实际登录页面地址

; 等待页面加载完成
While oIE.Busy
    Sleep, 100

; 输入账号和密码
Sleep, 1000
oIE.document.getElementById("username").value := "your_username"  ; 替换为实际账号框的 ID
oIE.document.getElementById("password").value := "your_password"  ; 替换为实际密码框的 ID

; 点击登录按钮
oIE.document.getElementById("login_button").click()  ; 替换为实际登录按钮的 ID

使用快捷键触发自动登录
-------------------
可以将自动登录功能绑定到一个快捷键上,方便随时触发.

; 按 Ctrl+Shift+L 触发自动登录
^+l::
    Run, "C:\Program Files (x86)\Tencent\QQ\Bin\QQ.exe"  ; 启动 QQ
    WinWait, QQ
    WinActivate, QQ
    Sleep, 2000
    Send, your_qq_account
    Send, {Tab}
    Send, your_qq_password
    Send, {Enter}
return
