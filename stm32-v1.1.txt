使用注意事项
---------------
1. 强制退出"openocd -f scripts/stm32f429igt6.cfg -c "set V8_SMP_DEBUG 1" -c "bindto 0.0.0.0" -c "gdb_port 3333" -c "telnet_port 4444" -c "tcl_port 5555" -c init -c 'reset halt' -c "add_script_search_dir scripts/""会导致openocd无法再次启动,需要将Jlink USB拔出再插.
2. 程序下载需要telnet到openocd的4444端口然后执行"script ll"进行下载
3. ARMCC和arm-none-eabi-gcc的区别主要体现在串口
4. 串口打印中文,字符集最好是GB2312.UTF-8用串口调试工具会乱码.

串口打印
---------------
1. 使用printf-4.0.0.tar.gz库,需要手动实现_putchar函数
--
void _putchar(char character)
{
	UART_TypeDef *base = g_uart_bases[UART_CONSOLE_INSTANCE];
	uint32_t tmp_d = character;
	tmp_d &= 0xff;
	uart_drv_fput_char(base, tmp_d);
}

2. 使用编译器库,需手动实现_write函数
--
int _write (int fd, char *pBuffer, int size)  
{  
	for (int i = 0; i < size; i++)  
	{  
		while((USART1->SR&0X40)==0);          //等待上一次串口数据发送完成  
		USART1->DR = (uint8_t) pBuffer[i];    //写DR,串口1将发送数据
	}  
	
	return size;  
}
