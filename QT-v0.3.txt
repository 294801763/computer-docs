项目模板
--------------
Application                               Qt应用程序,包含Qt Quick和普通窗口程序
Library                                   创建动态库,静态库,Qt Creator自身插件,Qt Quick扩展插件
其他项目                                  创建单元测试,子目录项目,QT设计师自定义控件
Non-Qt Project                            可以创建纯C语言或纯C++项目 
import Project                            从版本控制系统来管理软件项目,导入一些使用过的项目

Application - Qt Widgets Application      普通窗口模板,传统基于剖析的窗体界面程序
Application - Qt Console Application      控制台应用程序

QMainWindow                               基于主窗口类的应用程序,一般主要用于比较复杂的程序,还可以包括菜单栏,状态栏,工具栏及多个停靠的工具对话框
QWidget                                   最简单最基本的窗口程序,可以容纳很多个控件实现程序等功能
QDialog                                   基于对话框,对话框一般主要用于弹窗,也可以用于主界面显示,从QWidget继承过来的.

inline void CenterWidgets(QWidget *widget, QWidget *host = Q_NULLPTR) {
    if (!host)
        host = widget->parentWidget();

    if (host) {
        auto hostRect = host->geometry();
        widget->move(hostRect.center() - widget->rect().center());
    }
    else {
        QRect screenGeometry = QGuiApplication::screens()[0]->geometry();
        int x = (screenGeometry.width() - widget->width()) / 2;
        int y = (screenGeometry.height() - widget->height()) / 2;
        widget->move(x, y);
    }
}

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    Dialog w;
	
	CenterWidgets(&w);
	w.show();
	 
    return a.exec();
}

信号与槽机制(signal & slot)
--------------
信号(24中设计模式<观察者>):信号本质是事件,信号展现方式就是函数.
槽(slot):对信号的响应函数,槽就是一个函数;槽函数与普通函数的区别:槽函数可以与一个信号关联,当信号被发射时,关联的槽函数被自动执行处理.

信号函数只需要声明(不需要定义/实现),槽函数需要定义/实现.

const static double PI = 3.1415;

MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
    , ui(new Ui::MainWindow)
{
	this->setCentralWidget(new QWidget(this));

    l1 = new QLabel(tr("请输入圆的半径"),this);
    le = new QLineEdit(this);

    l2 = new QLabel(this);
    l2->setText(tr(""));

    btn = new QPushButton(this);
    btn->setText(tr("计算圆面积"));

    QGridLayout *layout = new QGridLayout(this);
    layout->addWidget(l1,0,0);
    layout->addWidget(le,0,1);
    layout->addWidget(l2,1,0);
    layout->addWidget(btn,1,1);

    this->centralWidget()->setLayout(layout);

    connect(le,SIGNAL(textChanged(QString)),this,SLOT(calcArea()));
}
...
void MainWindow::calcArea()
{
    bool isloop;
    int value;
    double area;

    QString tstr;
    QString vstr = le->text();

    value = vstr.toInt(&isloop);
    area = value * value * PI;
    l2->setText(tstr.setNum(area));
}
