自己建立reality节点
-------------------
step.1 安装xray
--
bash -c "./install" @ install --local

step.2 通过命令生成uuid
--
xray uuid > uuid

step.3 生成公和私密钥
--
xray x25519 > key

step.4 查看uuid和key
--
cat uuid
cat key

step.5 上传配置文件和修改配置文件
--
cd /usr/etc/xray/

step.6 config配置文件模板

step.7 清除\r
--
dos2unix /etc/xray/config.json

step.8 启动Xray命令
--
ufw allow 443
service xray start

step.9 重启Xray命令
--
service xray restart
